The 6 rules (that's all)

  • Protein first. Eggs, dahi, daal — eat these before filling up on roti.
  • 2 roti cap per meal. Roti is fine; the 4th roti is the belly.
  • Drinks don't get sugar. Water, lassi (namkeen), chai with less sugar. No soft drinks with meals.
  • Snack = fruit or nuts. Banana + a handful of peanuts/channa replaces biscuits and chips.
  • One planned junk meal a week. Enjoy it fully, zero guilt, and it's done. Planned junk is not failure — unplanned junk spirals.
  • Kitchen closes at 10 PM. Late-night eating on your sleep schedule is the silent belly-builder.

The system, in 30 seconds

  • Every day has 3 sizes: Full (~15 min), Lite (5 min), Spark (2 min). All three count.
  • Never miss twice. One missed day bends the chain (amber) — it doesn't break it. Two in a row and the chain rests; your comeback restarts it, and nothing else is lost.
  • Week won = any 4 active days.
  • Rotation: A (Push & Core) → B (Legs & Back) → C (Stamina), walks in between, Sunday hike.
  • Back pain: no sit-ups, no crunches. Dead bugs, bird dogs, bridges and planks build the same core while healing the lower back.

❤️ Apple Health sync — one-time setup

Apple only lets native apps talk to Health directly, so Sehat uses the official bridge: Apple Shortcuts. Build these once (2 min each) and the ❤️ buttons in the app come alive.

1 · "Sehat Log" — workouts → Health
  • Open the Shortcuts app → tap +
  • Rename it exactly: Sehat Log
  • Add action → search "Get Numbers from Input" → set its input to Shortcut Input. (Sehat sends the minutes as text — this converts it to a number. Skipping this step causes the "There was a problem" error.)
  • Add action → search "Log Workout"
  • Set Type: Functional Strength Training
  • Tap Duration → select variable → Numbers (from the step above), unit: minutes
  • Make sure "Show When Run" is OFF on the Log Workout action (tap the arrow on it)
  • Done. First run will ask for Health permission — allow it.
2 · "Sehat Pull" — steps & weight → Sehat
  • New shortcut, name it exactly: Sehat Pull
  • Add "Find Health Samples" → type Steps → filter: Start Date is today
  • Add "Calculate Statistics" → Sum of the samples
  • Add "Find Health Samples" → type Weight → Sort by Start Date, Latest First, Limit 1
  • Add "Text" action containing: {"steps": [Statistics], "weight": [Health Samples]} (insert the two variables)
  • Add "Copy to Clipboard"
  • Run it, then open Sehat → Safar → ❤️ Import. Bonus: 6,000+ steps auto-counts as your day's movement.
3 · "Sehat Weight" — weight → Health (optional)
  • New shortcut, name it exactly: Sehat Weight
  • Add action → "Log Health Sample" → type Weight
  • Tap Value → select variable → Shortcut Input, unit kg
  • Now the Safar tab's "Send weight to Health" button works.
